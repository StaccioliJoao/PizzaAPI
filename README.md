# PizzaAPI
Pizza API to simulate a pizza place order taking system

## Getting Started

### Dependencies

* fastapi
* pydantic
* uvicorn(optional)

### Running API

* How to run the api utiziling uvicorn
```
pip install fastapi
```
```
pip install pydantic
```
```
pip install uvicorn
```

```
uvicorn main:app --reload
```
